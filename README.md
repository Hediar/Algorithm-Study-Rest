### 정비기간 규칙

- 매주 1문제 이상 PR 해야합니다.(난이도 너무 낮은 것 인정x)

fork -> 각자 이름으로 폴더 생성 -> add/commit/push -> 일요일까지 PR -> 리뷰의 경우 Issue를 사용해주세요
